Homunkulus

Eine Backup-Lösung für einen Umfassenden Schutz

Datenverluste können jederzeit durch Computerfehlfunktionen, Viren oder Maleware verursacht werden. Mithilfe von Homunkulus ist es ein Kinderspiel, ein komplettes Backup Einer Festplatte oder ein Teilbackup ihrer wichtigsten Dateien zu erstellen. Stellen Sie sicher, dass Ihre Daten zu jeder Zeit geschützt sind.

Sicherung und Wiederherstellung von Festplatten

Sichern Sie alle Daten und Einstellungen einer Partition oder aller Partitionen (Festplatten), um im Falle einer Datenbeschädigung absolute Sicherheit zu haben.

Systemsicherung und Wiederherstellung

Sichern Sie Systeme, Boot-Partitionen und Programmdateien, um Ihr System einfach, effizient und sicher wiederherstellen zu können.

Sicherung und Wiederherstellung von Dateien

Sichern Sie Dateien auf separaten Partitionen auf lokalen Laufwerken oder auf externen Speichermedien, um einen umfassenden Schutz zu gewährleisten.

Einfache und Benutzerfreundliche Oberfläche

Durch unser Simples und einfaches Design ist die Bedienung von Homunkulus einfach und selbstverständlich. Es gibt keine Verschalteten Menus oder Komplizierten Tastenkombinationen. Bei Homunkulus tut jeder Button genau das, was auf ihn steht

Backup

Auf der Backup Seite können sie ihre Dateien, welche sie kopieren möchten, einfügen dies geschieht über das Obere Eingabe Feld. Durch das Betätigen des Add Path Button wird diese Eingabe bestätigt. Die eingebenden Pfade werden ihnen dann unten in der großen Anzeige angezeigt. Möchten sie nun ihre Eingabe rückgängig machen, so klicken sie auf den [Platzhalter] Button. Wenn sie keine Lust habe, die Pfade per Hand einzufügen, können sie auch .txt Dateien hochladen (Hierbei ist zu beachten das ALLES, was in dieser Datei steht, auch ausgelesen wird). Sind sie so weit fertig, können sie durch das drücken der großen gelben Buttons ganz unten das Backup Starten. Dieser Prozess kann ein stückle dauern. Wenn sie alles danach leeren wollen, können sie mit dem Clear Button alles wider Rückgängig machen.

Import

Unter dem Import Button finden sie ein übersichtliches Interface welches aus einer Großen Textbox und einem Button besteht. Durch den Klick auf den Button wird ein Explorer fenster geöffnet und sie können ganz bequem sich raussuchen von wo sie ihre daten Importieren möchten. Nach dem sie auf öffnen geklickt haben wird der Import automatisch ausgeführt

History

Die History Seite ist Relative schnell und Simpel erklärt, hier können sie ihre Statistiken sehen. Dieser wären zum Beispiel das Datum, wann das letzte Backup durchgeführt wurde. Oder wie lange das Backup gedauert hat. Das wichtigste ist aber der Name und der Pfad, wo das Backup gespeichert wurde
